# **App Name**: Isometric 21

## Core Features:

- AI Workout Image Tool: Utilize AI to dynamically generate high-quality visual aids for each isometric workout pose based on specific textual prompts, showcasing an athletic woman in a professional gym setting.
- Interactive Workout Timer: A prominent, clear countdown timer for exercise durations, inter-rep rests, and inter-set rests, featuring audible cues for countdowns and transitions.
- Customizable Intervals: Allow users to adjust the hold time for each exercise (30, 45, or 60 seconds), rest time between exercises (10, 20, or 30 seconds), and rest time between sets (20, 30, or 40 seconds).
- Workout Exercise Display: Display the current exercise with its AI-generated image, name, key instructions, and target muscles, ensuring a smooth flow between steps.
- 21-Day Challenge Tracker: Track the user's progress through the 21-day challenge, showing completed days and overall achievement.
- Responsive Interface: Provide a fully responsive and intuitive user interface optimized for a seamless experience across various device sizes (mobile, tablet, desktop).

## Style Guidelines:

- Primary color: #76CEDB (light blue-green) for interactive elements and highlights, suggesting freshness and focus in a modern gym environment. (HSL: 195, 50%, 65%)
- Background color: #1E2527 (very dark grey with a hint of blue-green) providing a professional, calm, and disciplined backdrop for the application. (HSL: 195, 20%, 15%)
- Accent color: #26DBAF (vibrant green-blue) for calls to action, progress indicators, and significant interactive components to provide clear visual cues. (HSL: 165, 70%, 50%)
- Body and headline font: 'Inter' (sans-serif) for its modern, neutral, and highly legible characteristics, ensuring clear communication of workout instructions and data.
- Utilize simple, clean, and line-based icons for workout controls, settings, and navigation to maintain a minimalist and modern aesthetic.
- Employ a clean and structured layout with a strong focus on the current exercise, timer, and visual guidance. Card-like components for settings and workout details will enhance usability and readability on all devices.
- Implement subtle, smooth transitions for timer updates, changes between exercises and rest periods, and other user interactions to create a fluid and engaging user experience.
